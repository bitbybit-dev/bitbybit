# Pack Multiple Product Variants into a Single glTF File

## Blender to 3D Bits Workflow for Product Variants

In this detailed tutorial, you will learn how to pack multiple product variants into a single glTF file and use it with the 3D Bits app for Shopify. Matas walks you through the latest features that streamline the parsing of various 3D nodes defined in Blender.

You will also learn how to use a new feature that automatically sets up the GUI directly from your Shopify product page. We've simplified several complex steps to help you build configurators faster and more efficiently.

Matas also shares practical advice on how to professionalize your workflow by using two sets of templates - one for draft products and another for live products. In addition, you'll learn how to use debug mode on your product page, along with other features supported in the latest version of the app.
